Redis

¿Qué es Redis?

Redis (Remote Dictionary Server) es una base de datos NoSQL en memoria orientada a pares clave-valor. Fue diseñada para ofrecer altísimo rendimiento y baja latencia, ya que guarda los datos principalmente en memoria RAM, aunque también permite persistirlos en disco. Es muy usada como caché, cola de mensajes, contador en tiempo real y almacenamiento temporal en aplicaciones que necesitan rapidez.

Principales características de Redis

◆⚡ Alto rendimiento: trabaja en memoria, lo que permite millones de operaciones por segundo.

◆🔑 Modelo clave-valor: los datos se almacenan como pares clave-valor.

◆🗂️ Tipos de datos avanzados: no solo cadenas, también listas, conjuntos, hashes, sorted sets, bitmaps, streams, etc.

◆🛠️ Operaciones atómicas: garantiza que ciertas operaciones se ejecuten de manera completa sin interferencias.

◆🌐 Ligero y rápido de implementar: muy usado en microservicios y entornos en la nube.
