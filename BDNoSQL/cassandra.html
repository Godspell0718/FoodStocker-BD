Cassandra

¿Qué es Cassandra DB?

Apache Cassandra es una base de datos NoSQL distribuida y de código abierto, diseñada para manejar grandes volúmenes de datos en múltiples servidores sin un único punto de fallo. Fue creada inicialmente en Facebook y hoy es un proyecto de la Apache Software Foundation. Cassandra utiliza un modelo basado en columnas anchas en lugar de documentos o tablas relacionales tradicionales, lo que la hace ideal para aplicaciones que necesitan alta escalabilidad y disponibilidad.

Principales características de Cassandra

◆📄 Modelo de datos basado en columnas: Organiza la información en tablas, filas y columnas, pero con un diseño flexible que no requiere un esquema rígido como en SQL.

◆📊 Escalabilidad horizontal masiva: Permite agregar fácilmente nuevos nodos al clúster para distribuir la carga y aumentar la capacidad.

◆🔄 Alta disponibilidad: No tiene un único punto de fallo; los datos se replican automáticamente entre múltiples nodos.

◆⚡ Rendimiento en escritura muy alto: Está optimizada para manejar flujos masivos de inserciones de datos.

◆ 🔍 Lenguaje de consultas (CQL): Usa Cassandra Query Language, similar al SQL, lo que facilita su aprendizaje.

Ejmeplo de Script en en Cassandra DB

// 🔹 CREATE - Insertar un usuario const insertQuery = 'INSERT INTO usuarios (id, nombre, edad, email) VALUES (?, ?, ?, ?)'; const id = cassandra.types.Uuid.random(); await client.execute(insertQuery, [id, 'Laura', 27, 'laura@example.com'], { prepare: true }); console.log(`🟢 Usuario insertado con id: ${id}`);

// 🔹 READ - Consultar usuario por id const selectQuery = 'SELECT * FROM usuarios WHERE id = ?'; const result = await client.execute(selectQuery, [id], { prepare: true }); console.log('📄 Usuario leído:', result.rows[0]);

// 🔹 UPDATE - Actualizar edad del usuario const updateQuery = 'UPDATE usuarios SET edad = ? WHERE id = ?'; await client.execute(updateQuery, [28, id], { prepare: true }); console.log('🟡 Usuario actualizado');

// 🔹 DELETE - Eliminar usuario const deleteQuery = 'DELETE FROM usuarios WHERE id = ?'; await client.execute(deleteQuery, [id], { prepare: true }); console.log('🔴 Usuario eliminado');
